# AI Disclosure

It would be foolish to discount the usefulness of the AI tools that are available to us today. LLMs can help quickly sort data, review logs and aide research.

This specific project started with human ideas, human made scripts. The intent remains human, though AI-assisted auto-complete has drastically changed how writing occurs. Intellisense has evolved to make the easy things like loops and ifs much more predictable. 

To that end, I still write the scripts. But auto-complete makes that job easier, and Claude certainly helps with review.
